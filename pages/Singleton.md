- The singleton pattern is a controlled instantiation pattern that guarantees a single instance of a class within a defined scope while providing a globally accessible well-governed access point to that instance.
	- KEY FEATURES:
		- It ensures consistency
		- Prevents thread race conditions
		- Majorly used in connection-pool, NestJS etc.
